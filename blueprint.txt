================================================================================
🎨 DESIGN SYSTEM
================================================================================

1. TEMA & WARNA
   - Mode             : Dark Mode
   - Background Utama : #0A0E21 (Deep Navy)
   - Surface/Card     : #1D1F33 (Dark Grey-Blue)
   - Primary Color    : #6C63FF (Purple/Violet)
   - Secondary Color  : #00D9FF (Cyan/Aqua)
   - Danger/Error     : #FF6B6B (Coral Red)
   - Success          : #4CAF50 (Green)
   - Warning          : #FFA726 (Orange)
   - Info             : #42A5F5 (Light Blue)

2. TIPOGRAFI
   - Font Family      : Roboto
   - Heading          : Bold, 20-36px
   - Body             : Regular/Medium, 12-16px
   - Caption          : 10-12px, White54/White38

3. KOMPONEN UI
   - Border Radius    : 12-24px (rounded modern)
   - Card Shadow      : Colored glow shadows (branded)
   - Gradient         : Linear gradient (Purple → Cyan / Purple → Dark Purple)
   - Animasi          : Fade, Scale, Slide, Pulse transitions
   - Status Bar       : Transparent, icon brightness light

1. SPLASH SCREEN (splash_screen.dart)
──────────────────────────────────────────────────────────────────────────────
   Tujuan    : Tampilan pembuka saat aplikasi pertama dibuka
   Durasi    : ~3.6 detik total sebelum navigasi ke Login

   Komponen:
   - Background gradient (Deep Navy → Dark Purple → Deep Navy)
   - Efek glow lingkaran (Purple & Cyan) di pojok atas-kanan & bawah-kiri
   - Logo ikon wallet dengan gradient (Purple → Cyan)
   - Teks "PayFlow" dengan efek ShaderMask gradient
   - Subtitle "Pembayaran Cepat & Aman"
   - Linear progress indicator
   - Teks versi "v1.0.0" di bagian bawah

   Animasi:
   - Logo: Scale (elasticOut) + Opacity fade-in (1200ms)
   - Teks: Fade-in + slide up (800ms, delay 800ms)
   - Pulse: Logo berdenyut 1.0 → 1.08 (1500ms, loop)
   - Transisi ke Login: FadeTransition (800ms)
